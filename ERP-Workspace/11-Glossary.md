## Domain Terms

| Term | Definition |
|------|-----------|
| AIDD | AI-Integrated Development Discipline -- guardrails framework for AI-driven operations ensuring human oversight for high-risk actions |
| Bounded Context | A DDD concept representing a distinct domain model boundary; ERP-Workspace has 11 bounded contexts |
| Breakout Room | A sub-room within a video meeting where a subset of participants can have a separate discussion |
| CalDAV | Calendar Distributed Authoring and Versioning -- an Internet standard for calendar synchronization (RFC 4791) |
| Canned Response | A pre-written email reply template that can be inserted with a keyboard shortcut |
| Channel | A persistent chat room organized around a topic, team, or project; can be public or private |
| CloudEvents | A specification for describing event data in a common way (CNCF project) |
| Collaboration Session | A real-time co-editing session where multiple users simultaneously edit a document |
| Conversation (Email) | A group of related email messages connected by In-Reply-To/References headers |
| Conversation (Chat) | A chat container that can be a channel, group DM, or 1:1 direct message |
| DKIM | DomainKeys Identified Mail -- email authentication method using cryptographic signatures |
| DLP | Data Loss Prevention -- policies and tools to prevent unauthorized sharing of sensitive information |
| DMARC | Domain-based Message Authentication, Reporting, and Conformance -- email authentication policy |
| Distribution Group | A mailing list that broadcasts messages to all members with optional moderation |
| eDiscovery | Electronic Discovery -- the process of identifying and collecting electronically stored information for legal proceedings |
| Focus Inbox | AI-powered email triage that separates important messages from noise |
| Free/Busy | Calendar availability information showing when a user is available, busy, or tentative |
| JMAP | JSON Meta Application Protocol -- modern email access protocol (RFC 8620/8621) replacing IMAP |
| Knowledge Graph | An entity-relationship network automatically extracted from email content, mapping people, organizations, and topics |
| LiveKit | An open-source WebRTC SFU (Selective Forwarding Unit) for real-time video/audio |
| MinIO | An S3-compatible object storage server used for file and blob storage |
| MTA | Mail Transfer Agent -- software that transfers email messages between servers using SMTP |
| MX Record | Mail Exchanger DNS record specifying the mail server responsible for a domain |
| Nextcloud | An open-source file sync and share platform providing WebDAV access and collaboration features |
| ONLYOFFICE | An open-source document editing suite supporting real-time collaborative editing of Office formats |
| Operational Transformation | An algorithm for real-time collaborative editing that resolves concurrent edit conflicts |
| PII | Personally Identifiable Information -- data that could identify a specific individual (SSN, credit card, etc.) |
| Public Folder | A shared folder accessible to multiple users within an organization, supporting mail, calendar, and contacts |
| Redpanda | A Kafka-compatible event streaming platform written in C++ (no JVM dependency) |
| RRULE | Recurrence Rule -- iCalendar standard for specifying event recurrence patterns (RFC 5545) |
| RTMP | Real-Time Messaging Protocol -- used for streaming video to platforms like YouTube and LinkedIn |
| RLS | Row-Level Security -- PostgreSQL feature enforcing tenant data isolation at the database level |
| S/MIME | Secure/Multipurpose Internet Mail Extensions -- email encryption and digital signing standard |
| SFU | Selective Forwarding Unit -- a video conferencing architecture that forwards media streams without transcoding |
| Shared Mailbox | An email mailbox accessed by multiple users with role-based permissions (reader, sender, admin) |
| Smart Compose | AI-powered email drafting assistance that suggests completions based on context |
| SPF | Sender Policy Framework -- email authentication method specifying authorized sending servers |
| Team Drive | A shared file storage space owned by a team rather than an individual |
| Triage Model | A per-user machine learning model that prioritizes incoming emails based on behavior patterns |
| Unified Inbox | A single interface consolidating email, chat, calendar, and file notifications |
| Waiting Room | A pre-meeting holding area where participants wait until the host admits them |
| WebDAV | Web Distributed Authoring and Versioning -- HTTP extension for file management |
| Webinar | A one-to-many presentation format with distinct presenter and attendee roles |
| WOPI | Web Application Open Platform Interface -- Microsoft protocol for integrating document editors |
