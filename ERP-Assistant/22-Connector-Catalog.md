# ERP-Assistant Connector Catalog

## 1. Overview

ERP-Assistant connects to 28+ data sources through four connector categories: ERP Internal (auto-discovered), Productivity (OAuth2), Communication (API), and Storage (SDK). Each connector implements the standard `Connector` interface and is managed by the connector-hub service.

## 2. ERP Internal Connectors

### Auto-Discovery Protocol

Internal connectors are auto-generated by scanning ERP modules for `capabilities.json`:

```mermaid
sequenceDiagram
    participant CH as connector-hub
    participant MOD as ERP Module

    loop Every 5 minutes
        CH->>MOD: GET /v1/capabilities
        MOD-->>CH: capabilities.json
        CH->>CH: Generate tool definitions
        CH->>CH: Update assistant-core tool registry
    end
```

## 6. Connector Health Status

```mermaid
stateDiagram-v2
    [*] --> Discovered
    Discovered --> Initialized: Config loaded
    Initialized --> Connecting: User initiates OAuth
    Connecting --> Connected: Token received
    Connected --> Healthy: Health check passes
    Healthy --> Degraded: Intermittent errors
    Degraded --> Healthy: Errors resolve
    Degraded --> Unhealthy: Persistent failures
    Unhealthy --> Connecting: Re-authenticate
    Connected --> Disconnected: User disconnects
    Disconnected --> [*]
```
